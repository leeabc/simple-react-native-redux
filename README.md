# Simple React Native with Redux

## Include

- React Native
  
- Redux
  
- React-Redux **(Cannot use ver 4 !!)**
  
- Redux Middleware
  
  - redux-thunk
  - redux-logger
  
- Easy message example
  
- Fetch weather example (Async)
  
  

## Start to try React-Native with Redux

- npm install
  
- open iOS / Android project
  
- Run project
